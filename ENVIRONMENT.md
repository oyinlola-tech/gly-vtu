# Environment Variables

This project reads environment variables from `.env`. Use `.env.example` as a template.

## Server
- `PORT`
- `NODE_ENV`
- `TRUST_PROXY`
- `CORS_ORIGIN`

## Database
- `DB_HOST`
- `DB_PORT`
- `DB_USER`
- `DB_PASSWORD`
- `DB_NAME`

## Auth and Cookies
- `JWT_SECRET`
- `JWT_ADMIN_SECRET`
- `JWT_ACCESS_TTL`
- `JWT_REFRESH_DAYS`
- `COOKIE_REFRESH`
- `COOKIE_ENC_SECRET`

## Rate Limits
- `RATE_LIMIT_GLOBAL_MAX`
- `RATE_LIMIT_AUTH_MAX`
- `RATE_LIMIT_ADMIN_AUTH_MAX`
- `RATE_LIMIT_OTP_MAX`
- `RATE_LIMIT_WEBHOOK_MAX`
- `RATE_LIMIT_BILLS_MAX`

## Email (SMTP)
- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_SECURE`
- `SMTP_USER`
- `SMTP_PASS`
- `EMAIL_FROM`
- `EMAIL_LOGO_URL`
- `BRAND_URL`
- `SUPPORT_URL`
- `PRIVACY_URL`
- `TERMS_URL`

## Security Questions and PINs
- `OTP_COOLDOWN_SECONDS`
- `OTP_MAX_PER_HOUR`
- `PIN_MAX_ATTEMPTS`
- `PIN_LOCK_MINUTES`
- `SECURITY_QUESTION_REQUIRE_PIN_CHANGE`
- `SECURITY_QUESTION_REQUIRE_DEVICE_VERIFY`
- `SECURITY_QUESTION_REQUIRE_PASSWORD_RESET`
- `SECURITY_QUESTION_TRANSFER_LIMIT`
